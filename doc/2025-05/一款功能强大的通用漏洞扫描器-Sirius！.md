## 0x01 工具介绍  
  
Sirius 是一款开源通用漏洞扫描器，利用社区驱动的安全情报。通过基于 Docker 的设置，几分钟内即可开始使用。## 0x02 安装  
  
**1.先决条件**  
1. Docker Engine 20.10.0+  
  
1. Docker Compose V2  
  
1. 至少 4GB 内存  
  
1. 至少 10GB 空闲磁盘空间  
  
**2.安装**  
```
# 克隆仓库
git clone git@github.com:SiriusScan/website.git
cd website
# 启动所有服务
docker compose up -d
# 访问 Web 界面
open http://localhost:3000
```  
  
3.登录  
```
用户名：admin  密码：password
```
